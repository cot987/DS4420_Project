# Project Topic: Testing the “Hot Hand” in the NBA Using Time Series and Bayesian Models

Within the "NBA Shot Data" folder you will find the data collected that we will be using for this project. The data is gathered from Kaggle, and includes the data regarding every shot in every NBA game from 2004 to 2024. This includes information like what game the shot was made in, who made the shot attempt, and whether the shot went in or not.

Within the "Proof of Concept" coding file, you will find the beginning of the Time Series model we will be using for analyzing this data. We run a basic time series model on a subset of the data years, since running all years would take a long time, to prove that running a time series is possible. We then show the highest correlations for the top 5 shooters, and include PACF and ACF plots for said individuals.
